# Overview

The JavaScript language as a long list of operators that fall into the following categories:

* Assignment
* Comparison
* Arithmetic
* Logical
* String \(covered in Strings section\)
* Conditional \(ternary\)
* Bitwise
* Comma operator
* Unary operators
* Relational operators

MDN has the complete lists.
